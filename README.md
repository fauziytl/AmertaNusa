AmertaNUSA adalah aplikasi web reservasi restoran yang dirancang untuk memberikan pengalaman pemesanan meja yang cepat, praktis, dan modern bagi pelanggan. Sistem ini menyediakan alur reservasi lengkap mulai dari melihat menu, memilih meja yang tersedia, mengisi formulir reservasi, melakukan pembayaran, hingga memeriksa status pemesanan.
Website ini dibangun menggunakan HTML, CSS, JavaScript, serta PHP sebagai backend untuk pengecekan ketersediaan meja dan status reservasi. Desainnya menggunakan konsep warna elegan dengan sentuhan biru gelap dan oranye khas AmertaNUSA untuk memberikan kesan modern dan profesional.
